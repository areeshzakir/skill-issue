Paperclip provides a command interface for interacting with the Paperclip control plane, enabling you to manage tasks, orchestrate agents, monitor running workflows, and enforce governance policies across your AI agent fleet. It serves as the central hub for dispatching work, checking agent status, reviewing task outputs, and managing the lifecycle of autonomous operations.

Use Paperclip when you need to create, assign, or monitor tasks across multiple agents, when you want to check the status of running workflows, or when you need to enforce governance rules on agent behavior. It is the primary interface for anyone operating within the Paperclip ecosystem, whether managing a single agent or coordinating a fleet of specialized workers.

Example: Run Paperclip to list all active agents and their current tasks, create a new task "Audit Q1 marketing spend" and assign it to the analytics agent, check the governance log for any policy violations in the last 24 hours, and review the output queue for completed tasks awaiting approval.
