Colorize introduces a thoughtful color system to interfaces that are currently monochromatic, grayscale, or under-utilizing color as a communication tool. It generates a cohesive palette with primary, secondary, and accent colors, then applies them strategically to reinforce hierarchy, indicate interactive elements, communicate status, and create visual rhythm across the interface.

Use Colorize when your interface feels flat or clinical due to an over-reliance on grays and whites, or when you need to introduce semantic color coding for status indicators, categories, or data visualization. It ensures all color choices meet WCAG contrast requirements and work for users with color vision deficiencies.

Example: You have an admin dashboard rendered entirely in shades of gray with blue links. Run Colorize to introduce a warm accent palette where success states use green, warnings use amber, destructive actions use red, and interactive elements use a distinctive indigo. The skill generates CSS custom properties and a dark mode variant automatically.
