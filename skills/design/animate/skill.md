Animate adds purposeful motion design to your interfaces, transforming static screens into fluid, responsive experiences. It identifies opportunities for meaningful animations such as page transitions, loading states, hover effects, scroll-triggered reveals, and component entrance/exit animations. All motion suggestions follow the principle that animation should communicate state changes and guide attention, never distract.

Use Animate when your interface feels flat or when users need visual cues to understand what changed after an interaction. It is especially valuable for onboarding flows, form submissions, navigation transitions, and data visualizations where motion can clarify relationships between elements.

Example: You have a settings page where toggling a switch saves preferences silently. Run Animate to add a subtle confirmation ripple on the toggle, a brief success checkmark animation, and smooth height transitions when conditional sections expand or collapse. The skill generates production-ready Framer Motion or CSS animation code.
