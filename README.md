# skill-issue

> "Sounds like a skill issue." — Now it literally is.

A curated collection of AI agent skills accumulated from various users, developers, and organizations. One dashboard to find the right skill (or group of skills) for any task or goal.

## What's This?

Instead of hunting through scattered repos, configs, and forgotten bookmarks, `skill-issue` is a single place to catalog, organize, and discover skills for AI coding agents (Claude Code, Codex, Gemini CLI, etc).

## Structure

```
skills/
  <category>/
    <skill-name>/
      skill.md          # The skill definition
      README.md         # Usage notes, examples, gotchas
```

### Categories

| Category | Description |
|----------|-------------|
| `engineering` | Code generation, debugging, testing, refactoring |
| `design` | UI/UX, frontend design, animation, accessibility |
| `devops` | CI/CD, deployment, infrastructure, monitoring |
| `marketing` | Ads, SEO, content, social media |
| `productivity` | Workflow automation, project management, planning |
| `research` | Web research, analysis, documentation |
| `meta` | Skills about skills — creation, evaluation, optimization |

## Adding a Skill

1. Pick the right category (or propose a new one).
2. Create a directory under `skills/<category>/<skill-name>/`.
3. Add `skill.md` with the skill definition.
4. Add a `README.md` with usage examples and notes.
5. Submit a PR.

## Why "skill-issue"?

Because every problem is a skill issue — you just haven't found the right skill yet.
